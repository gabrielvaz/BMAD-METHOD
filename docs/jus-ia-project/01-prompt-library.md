# Ideia 01: Prompt Library Curada por Área do Direito

> Biblioteca de prompts testados e validados por advogados especialistas, organizados por área do direito e tipo de tarefa.

## Status: Aguardando Priorização

## Dor que Resolve

- Advogados não sabem formular prompts eficazes (79% nunca se capacitaram)
- Conteúdo em PT-BR sobre IA jurídica é escasso
- Gap entre "ter acesso à ferramenta" e "extrair valor real"

## Conceito

Biblioteca organizada de prompts jurídicos prontos para uso, testados em ferramentas como Jus IA e ChatGPT, com ratings da comunidade e exemplos de output esperado.

## Estrutura Proposta

```
📚 Prompt Library
├── 🏛️ Direito Civil
│   ├── Petição Inicial (danos morais, cobrança, despejo...)
│   ├── Contestação
│   ├── Recursos (apelação, agravo, embargos)
│   └── Contratos (locação, prestação de serviço...)
├── ⚖️ Direito Trabalhista
│   ├── Reclamação Trabalhista
│   ├── Cálculos (verbas rescisórias, horas extras)
│   └── Acordos
├── 🔒 Direito Penal
├── 💰 Direito Tributário
├── 👨‍👩‍👧 Direito de Família
└── 🏢 Direito Empresarial
```

## Anatomia de Cada Prompt

Cada prompt na biblioteca contém:

- **Template com variáveis** — `[nome do cliente]`, `[valor da causa]`, `[tribunal]`
- **Nível de complexidade** — Iniciante / Intermediário / Avançado
- **Rating da comunidade** — Estrelas + comentários de quem usou
- **Compatibilidade** — Versão otimizada para Jus IA vs. ChatGPT vs. Claude
- **Exemplo de output** — Resultado esperado para referência
- **Dicas de personalização** — Como adaptar para casos específicos

## Diferenciais

- Não é só uma lista — cada prompt é **testado, versionado e avaliado**
- Organizado por **área do direito + tipo de tarefa** (não genérico)
- Inclui **contexto legal** (artigos, súmulas, jurisprudência relevante)
- Atualizado conforme mudanças legislativas

## Público-Alvo Principal

- Advogados que já assinam Jus IA mas não extraem o máximo
- Advogados iniciantes em IA que precisam de ponto de partida
- Escritórios que querem padronizar o uso de IA na equipe
